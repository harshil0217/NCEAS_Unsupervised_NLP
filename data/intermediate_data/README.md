# Intermediate Data

This folder holds outputs generated by the pipeline. Most files are excluded from version control (see `.gitignore`); only the summary figures are tracked.

| Subfolder | Contents | Tracked? |
|-----------|----------|----------|
| `Qwen/` | Embeddings from `Qwen3-Embedding-0.6B` (`.npy` files, one per dataset) | No |
| `sentence-transformers/` | Embeddings from `all-MiniLM-L6-v2` (`.npy` files, one per dataset) | No |
| `clustering_results/` | Dimensionality reduction + clustering outputs (`.pkl` / `.npy` files) | No |
| `summary_figures/` | Final scatter grid and Shepard diagram PNGs committed to the repo | Yes |

To regenerate all intermediate data, run the benchmark pipeline as described in [INSTALL.md](../../INSTALL.md).
