# Intermediate Data

This folder holds outputs generated by the pipeline. All files are excluded from version control (see `.gitignore`).

| Subfolder | Contents |
|-----------|----------|
| `Qwen/` | Embeddings from `Qwen3-Embedding-0.6B` (`.npy` files, one per dataset) |
| `sentence-transformers/` | Embeddings from `all-MiniLM-L6-v2` (`.npy` files, one per dataset) |
| `clustering_results/` | Dimensionality reduction + clustering outputs (`.pkl` / `.npy` files) |

Final figures (scatter grids, Shepard diagrams) are saved to `results/`. To regenerate all intermediate data, run the benchmark pipeline as described in [INSTALL.md](../../INSTALL.md).
