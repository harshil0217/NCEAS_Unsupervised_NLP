# Results

This folder contains all pipeline outputs. Files are organized by output type.

## Folder Structure

```
results/
├── viz_metrics/                        # Visualization quality metrics (CSV)
│   ├── Qwen/                           # Qwen3-Embedding-0.6B
│   │   └── viz_metrics_{dataset}.csv
│   └── sentence-transformers/          # all-MiniLM-L6-v2
│       └── viz_metrics_{dataset}.csv
│
├── shepard_diagrams/                   # Shepard diagram PNGs
│   ├── Qwen/
│   └── sentence-transformers/
│
└── summary_figures/                    # Scatter grid PNGs (synthetic data)
```

Clustering score CSVs live directly in `results/`:
- Benchmark: `{dataset}_clustering_scores.csv`, `{dataset}_herc_*.csv`
- Synthetic: `{theme}_hierarchy_{config}_clustering_scores.csv`

## What Goes Where

| Output | Location | Generated by |
|--------|----------|--------------|
| Benchmark clustering scores | `results/` | `eval_pipeline.py` |
| Synthetic clustering scores | `results/` | `synthetic_eval_pipeline.py` |
| Hercules scores | `results/` | `herc_pipeline.py` / `synth_herc_pipeline.py` |
| Viz metrics CSVs | `results/viz_metrics/{provider}/` | `viz_metrics_script.py` |
| Shepard diagrams | `results/shepard_diagrams/{provider}/` | `viz_metrics_script.py` |
| Scatter grids | `results/summary_figures/` | `scatter_grid_synthetic.py` |

## Notes

- PNG files (Shepard diagrams, scatter grids) are gitignored — not tracked in git
- CSV files in `results/` and `results/viz_metrics/` are tracked
- `{provider}` is either `Qwen` or `sentence-transformers`
