# RCV1-v2 Dataset

RCV1-v2 (Reuters Corpus Volume 1, Version 2) is a standard benchmark for
hierarchical, multi-label text classification.

---

## Official Dataset Distribution

Linguistic Data Consortium (LDC):  
https://catalog.ldc.upenn.edu/LDC2005T04

> Note: Access typically requires institutional credentials.  
> This is normal and expected for this benchmark.

---

## Defining Reference Paper

The authoritative paper defining the dataset, hierarchy, and labels is:

Lewis et al. (2004), *Journal of Machine Learning Research*  
https://jmlr.csail.mit.edu/papers/volume5/lewis04a/lewis04a.pdf

This paper is the **required citation** for Phase 1.1, even if the data is accessed elsewhere.

---

## Repository Policy

Due to licensing restrictions, the raw dataset is **not stored in this repository**.

Team members with institutional access should download the dataset locally and
place it in this directory for experimentation in later phases.
